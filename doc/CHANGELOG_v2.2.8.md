# v2.2.8 更新日志

## 🐛 修复

### 横排渲染字序错乱

- 修复横排文本渲染时有概率出现字序错乱（最后一个字跑到第一个位置）的问题。
- 根因：`_line_surface` 中将 `QTextLayout.glyphRuns()` 所有 run 的 positions 简单展平拼接，但 Qt 对混合脚本文本（如 CJK + ASCII）会拆分为多个 font run，**run 的返回顺序不保证与字符串中字符的逻辑顺序一致**，导致位置与字符错位。
- 修复方式：按 x 坐标对 positions 排序后再与字符逐一配对，LTR 文本升序、RTL 文本降序。
